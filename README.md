# Blogging System Project
Python Blogging System Project as part of MS SWDV630 course

# Installation
- Install Latest python.
- Install Flask.
```
pip install flask
```

- Install SQL Alchemy.
```
pip install sqlchemy
```

- Install Flask Bootstrap
```
pip install flask-bootstrap
```
