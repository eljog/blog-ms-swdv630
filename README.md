# Blogging System Project
Python Blogging System Project as part of MS SWDV630 course

# Installation
- Install Latest python, Flask, SQL Alchemy and Flask Bootstrap
```
pip install flask
pip install sqlalchemy
pip install flask-bootstrap
```

- Run app

Set an environment variable `SECRET_KEY` with a unique secret value.

Run `python app.py`
